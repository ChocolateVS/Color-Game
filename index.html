How to play

Gameplay Objective

The aim of the game is to figure out a sequence of colors

Start by making a guess

Make a guess by selecting a cell and then selecting a color to apply to the cell. Once you fill in a row (complete your guess) click ok to submit the guess.

TIP: The first cell is selected by default, select a color to apply it directly. After applying the color, the selected cell will move accross

Normal Mode - Highlights

In normal mode, each time you make a guess cells are highlighted to show you whether the color you have guessed is:

In the correct position

In the sequence of colors but in the wrong position

Not in the sequence

Use this information to figure out which colors belong in each position and solve the sequence.

TIP: You can turn all highlights on/off in the options menu. You can turn on/off only the red highlights for a cleaner look

Classic Mode

In Classic mode, you are only given the number of correct colors, and colors in incorrect postions

Therefore, highlights are disabled.

In the following guess we can determine the following:
- There are two colors in the correct position (the number next to the green circle)
- There and two colors that are in the sequence but in an incorrect position (the number next to the blue circle)

Hard Mode

NOTE: NOT IMPLEMENTED

Hard Mode is basically the same as normal mode. The only difference is that once you discover a color's position, you must use it in your following guesses

Options

Number of guesses - Sets the number of guesses you have to guess the sequence

Guess Size - Set's how many colors in a guess

Number of colors - Set the number of colors that are selected at random, and could be part of the sequence

Guess Stat - Displays the Guess Stats (Classic mode counters)

Hide Incorect Guesses - Recommended for Normal and Hard Mode. Not Recommended for Classic mode
